# TSyper

Tsyper is a TUI app that test your typing speed with a uniqe random quotes

## Install

```
git clone https://github.com/MohamadSafi/TSyper.git
cd TSyper
python3 TSyper.py
```

## Options

For short quotes(default):
```
python3 TSyper.py short
```
For long quotes:
```
pyhton3 TSyper.py long
```
